Collection of scripts for world-of-dungeons.net

favmenu.user.js          - Allows to alter main menu by injecting customizable entries

herolist.user.js         - Adds functionality to order your heroes

profile_export.user.js   - Adds functionality to export profile info to BBCode

storage.user.js          - Adds additional functionality for group storage and treasure vault management

trade.user.js            - Sorts trade items and calculates total sum of uses
